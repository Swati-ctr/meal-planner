# 🍽️ AI Meal Planner

A full-stack AI-powered Meal Planner application that helps users create personalized weekly meal plans, generate grocery lists, and receive intelligent meal recommendations based on preferences, dietary restrictions, and different life situations.

## 🚀 Live Demo

🌐 **Frontend:** https://meal-planner-liard-tau.vercel.app/

## ✨ Features

* Secure JWT Authentication (Register & Login)
* AI-powered meal recommendations
* Personalized weekly meal planning
* Random meal generation
* Automatic grocery list generation
* Save and manage meal plans
* Share meal plans via WhatsApp
* Save cook details for quick sharing
* Voice assistance for meal plans
* Allergy preference support
* Responsive design for desktop and mobile

##  Tech Stack

### Frontend

* React.js
* HTML
* CSS
* Axios

### Backend

* Node.js
* Express.js
* JWT Authentication
* REST APIs

### Database

* MongoDB
* Mongoose

### Deployment

* Frontend: Vercel
* Backend: Render

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/Swati-ctr/meal-planner.git
```

### Install Frontend

```bash
cd cook-planner
npm install
npm run dev
```

### Install Backend

```bash
cd ../cook-planner-server
npm install
npm run dev
```
